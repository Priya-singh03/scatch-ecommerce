# Scatch - E-commerce Bag Store

Full-stack e-commerce web application built with Node.js, Express, and MongoDB. Users can browse products, add items to cart, and manage their account. Includes a separate owner/admin panel to add and manage products.

## Features
- User authentication (register, login, logout) with session-based auth
- Product listing with dynamic filtering and sorting
- Shopping cart — add, remove, increment, decrement quantity
- Real-time price calculation with grand total
- Owner/Admin panel to create new products with image upload
- Flash messages for user feedback

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Templating:** EJS
- **Styling:** Tailwind CSS
- **Auth:** Express-session, bcrypt
